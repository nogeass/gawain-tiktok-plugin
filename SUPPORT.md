# Support

## Getting Help

- **Documentation**: See the `docs/` directory and `README.md`
- **Issues**: [GitHub Issues](https://github.com/nogeass/gawain-tiktok-plugin/issues)
- **Gawain Platform**: [gawain.nogeass.com](https://gawain.nogeass.com)

## Reporting Bugs

When reporting a bug, please include:
1. Plugin version (`npm list gawain-tiktok-plugin`)
2. Node.js version (`node --version`)
3. Steps to reproduce
4. Expected vs actual behavior
5. Relevant error messages or logs
