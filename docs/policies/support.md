## Common Requests

### Disconnect My TikTok Shop Account

Use the disconnect feature in the app, or send a request to our support email with subject "Disconnect Request" and your install ID.

### Delete My Data

All data deletion requests are processed within 5 business days.

### Report a Bug

Email **support@nogeass.com** with:
- Description of the issue
- Steps to reproduce (if applicable)
- Screenshots (if applicable)

### Security Vulnerability

If you discover a security vulnerability, please report it responsibly:
- Email: **security@nogeass.com**
- Do not disclose vulnerabilities publicly before they are resolved.
